# TaskManager API

ASP.NET Core Web API project for task management.

## Features
- CRUD operations
- Filtering
- Pagination
- Sorting
- DTOs
- AutoMapper
- Entity Framework Core
- Swagger UI

## Technologies
- ASP.NET Core
- Entity Framework Core
- SQL Server
- AutoMapper
- Swagger

## Example endpoint
GET /api/tasks/filter?search=milk&page=1&pageSize=10
